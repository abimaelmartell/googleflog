Acerca de

Dada la moda que tienen algunas personas en internet de escribir cómo si les hubieran cortado los dedos y se los hubieran pegado al revés, surge esta parodia de Google Traductor para traducir de idioma español al idioma que utilizan en la red social metroflog.


Proyecto realizado por @Kinduff en colaboración de @frgii.

API

La API funciona de modo I/O y es muy sencilla; a través de la siguiente URL se devuelve un string transformado.

http://abarcarodriguez.com/googleflog/api.php?s=ejemplo
  => "** eEjeEmpl0o -- <3"

Puedes ver el código en Github. Pull requests e Issues son bienvenidos.
